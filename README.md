# ⚠️ DEPRECATED: AriA Minecraft Server Windows Client

> **Notice:** This desktop application is no longer in use or maintained.

The server infrastructure has been modernized. Starting and connecting to the server is now handled entirely through the **AriA Panel** (a web application).

Please see the main infrastructure repository for instructions on how to use the web panel, download mods, and connect to the server:
👉 **[aria-minecraft-server-iac](https://github.com/axiomeye/aria-minecraft-server-iac)**

### Legacy Build Instructions
*(For archival purposes only)*

**To run:**
```bash
dotnet run
```

**To export:**
```bash
dotnet publish --runtime win-x64 --configuration Release
```
